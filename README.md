# Framer Motion with React + Vite 🚀

This repository contains **all my Framer Motion animation files** created using **React.js and Vite**.  
It covers **basic to advanced animation concepts**, useful for learning, practice, and real-world UI animation implementation.

---

## 🛠 Tech Stack

- ⚛️ React.js
- ⚡ Vite
- 🎞 Framer Motion
- 🎨 CSS / Tailwind CSS (if used)

---

## 🚀 Getting Started

### 1️⃣ Clone the Repository

git clone https://github.com/Rudranil-12/Framer-Motion.git

### 2️⃣ Navigate to Project Folder

cd Framer-Motion

### 3️⃣ Install Dependencies

npm install

### 4️⃣ Run Development Server

npm run dev

---

## 📦 Framer Motion Installation

npm install motion
